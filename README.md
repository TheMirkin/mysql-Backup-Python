# mysql-Backup-Python


mysql yedekleme için Database Ayarlarını json dosyasından kendinize göre düzenleyin : 
```json
{
  "db": {
    "host": "localhost",
    "user": "root",
    "pass": "password",
    "dbname": "dbname",
    "file": "dosya.sql"
  }
}
```

Kullanımı : 
```
python Backup.py
```
